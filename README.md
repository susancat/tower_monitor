TOWER token corculation monitoring API

## install dependencies

```bash
npm i
# or
yarn install
```

## run the server:

```bash
npm run dev
# or
yarn dev
```

## environment variables
```bash
BASE_URL

ETH_ADDRESS
POLYGON_ADDRESS

POLYGON_RPC
ETH_RPC
TOTAL_SUPPLY
```
